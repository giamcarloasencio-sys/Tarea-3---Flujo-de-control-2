# Actividad 3 - Flujo de Control Parte 2

## Descripción

Este programa fue desarrollado en C++ y permite registrar las calificaciones de una cantidad n de estudiantes.

El programa:

- Solicita la cantidad de estudiantes.
- Registra el nombre y las cuatro calificaciones de cada estudiante.
- Calcula el promedio.
- Determina si el estudiante aprobó o reprobó.
- Muestra los resultados en una tabla organizada.
